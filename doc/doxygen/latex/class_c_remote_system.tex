Listens for new connections and creates new clients when a new connection occurs. 

\begin{DoxyReturn}{Returns}
none
\end{DoxyReturn}
Each time a new connection is done, a thread is created executing t\+Recv\+Type. This waits for receival of TYPE command, which identifies the type of the connecting remote client. When the type is defined, the client is added to the remote client list and the thread terminates.

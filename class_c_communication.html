Waits for thread termination (default action), or detaches from them.

Parameters
async- from asynchronous - defines if this function will be blocking (waiting for thread termination) or not (detaching)
Returns
none

Async can be used with defines 'RUN_BLOCK' and 'RUN_NONBLOCK'. If async=RUN_BLOCK the function will join the threads, waiting for their termination. If async=RUN_NONBLOCK the function will detach from the threads.
